# Project Final Paper

This repository contains the final paper, presentation materials, and the Python scripts used in the project.

## Demo Video
YouTube: https://youtu.be/ZUixDtCBw5w

## Repository Structure
- `src/` : Python scripts
- `data/` : dataset files used by the scripts
- `docs/` : report and presentation files (PDF/DOCX/PPTX)
- `media/` : local media (ignored by git)

## Run (example)
1. Install dependencies:
   - `pip install -r requirements.txt`
2. Run a script:
   - `python src/Main9.py`

> Note: If a script uses a hard-coded local path, update it to a relative path under `data/`.
